# Level Play Digital Brand Guide

A comprehensive web-based brand guide for Level Play Digital, designed for freelancers and partners creating social media assets.

## Overview

This brand guide provides complete brand guidelines including colors, typography, logo usage, imagery guidelines, social media templates, brand voice, spacing guidelines, and downloadable assets.

## Features

- **Interactive Color Palette**: Click color swatches or buttons to copy color codes
- **Comprehensive Typography Guide**: Complete font hierarchy and social media text guidelines
- **Logo Usage Guidelines**: Clear space requirements, minimum sizes, and do's/don'ts
- **Social Media Templates**: Visual templates for Instagram, Facebook, Twitter/X, and LinkedIn
- **Brand Voice & Tone**: Messaging guidelines and example copy
- **Spacing & Layout**: Grid system and spacing guidelines
- **Downloadable Assets**: Logo files and brand resources

## Usage

### Local Development

1. Open `index.html` in a web browser
2. No build step or server required - it's a static site
3. Use a local server for best experience:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (if you have http-server installed)
   npx http-server
   
   # Using PHP
   php -S localhost:8000
   ```

### Deployment

This is a static website that can be deployed to any static hosting service:

- **GitHub Pages**: ✅ Live at [https://steemer473.github.io/levelplaydigital-brand-guide/](https://steemer473.github.io/levelplaydigital-brand-guide/)
- **Netlify**: Drag and drop the folder or connect via Git
- **Vercel**: Connect your repository
- **Any web server**: Upload all files to your web server

### GitHub Pages

The brand guide is hosted on GitHub Pages. To update:
1. Make changes to files
2. Commit: `git add . && git commit -m "Update description"`
3. Push: `git push`
4. Changes will be live within a few minutes
