# Social Media Templates

This directory contains template files for creating social media assets.

## Template Formats

Templates are provided in various formats:
- **PSD** - Adobe Photoshop files with layers
- **Figma** - Figma design files
- **PNG** - Raster templates with guides
- **SVG** - Vector templates

## Platform-Specific Templates

### Instagram
- Square Post (1080 × 1080px)
- Story (1080 × 1920px)
- Reel (1080 × 1920px)

### Facebook
- Post (1200 × 630px)
- Cover (1640 × 859px)

### Twitter/X
- Post (1200 × 675px)
- Header (1500 × 500px)

### LinkedIn
- Post (1200 × 627px)
- Banner (1584 × 396px)

## Usage Guidelines

1. Always maintain logo clear space requirements
2. Keep important text in the safe zone (center 60-75%)
3. Use brand colors from the color palette
4. Follow typography guidelines
5. Export at the specified dimensions for each platform

## Creating Templates

If you need to create templates:
1. Use the dimensions specified in the brand guide
2. Include guides for safe zones
3. Place logo in designated area (bottom-right corner)
4. Use brand colors and fonts
5. Save in multiple formats for different tools
