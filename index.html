Body Text Styles

Use for important paragraphs and introductions. This size provides excellent readability while maintaining visual hierarchy.

Body Text - 16px / Regular (400) / Line-height: 24px / Color: #020817

This is the standard size for most content. It balances readability with content density, making it ideal for longer paragraphs and descriptions.

Small Text - 14px / Regular (400) / Line-height: 20px / Color: #64748B

Use for captions, fine print, and secondary information. Ensure sufficient contrast when using smaller text sizes.

Caption - 12px / Regular (400) / Line-height: 16px / Color: #64748B

Use for image captions and very small supporting text.

Button Usage Guidelines

Primary Buttons

  • Use gradient buttons for main CTAs (Get Started, Schedule Consultation)
  • Button text should always be white on gradient backgrounds
  • Use larger size (18px) for hero sections and prominent CTAs
  • Standard size (14px) for secondary CTAs and forms

Outline Buttons

  • Use for secondary actions (Explore, Learn More)
  • Cyan outline for primary secondary actions
  • Coral outline for alternative secondary actions
  • On hover, fill with background color and change text to white

Text Links

  • Always include right-pointing arrow icon (‚Üí)
  • Use Coral color (#FF6633) by default
  • Arrow should animate on hover (slides right)
  • Use for navigation links and "Learn More" actions

General Guidelines

  • Border-radius: 8px standard for all buttons
  • Maintain consistent padding ratios
  • Ensure sufficient contrast for accessibility
  • Use hover states to provide visual feedback
  • Keep button text concise and action-oriented

Logo Usage

The Level Play Digital logo is a key element of our brand identity. Follow these guidelines to ensure proper usage.

Primary Logo

Full Logo - Gradient Version

Clear Space

The logo requires clear space around it equal to the height of the "L" in "Level". This ensures the logo is never cramped or visually overwhelmed by other elements.

Minimum Sizes

  • Digital: Minimum 120px width for web use
  • Social Media: Minimum 200px width for profile images
  • Print: Minimum 1 inch width for printed materials

Logo Placement Guidelines

Social Media Profile Images

  • Center the logo in the profile image
  • Maintain clear space around all edges
  • Use high-resolution version (SVG preferred)

Post Graphics

  • Place logo in bottom-right or top-left corner
  • Use white or dark version based on background
  • Maintain minimum size requirements

Stories & Reels

  • Place logo subtly in corner
  • Ensure it doesn't obstruct important content
  • Use semi-transparent version if needed

Do's and Don'ts

‚úì Do

  • Use the logo at appropriate sizes
  • Maintain clear space around the logo
  • Use high-resolution files
  • Keep logo colors consistent
  • Place logo on contrasting backgrounds

‚úó Don't

  • Stretch or distort the logo
  • Change logo colors
  • Place logo on busy backgrounds
  • Rotate or tilt the logo
  • Add effects or filters to the logo
  • Use logo smaller than minimum sizes

Usage Guidelines

  • Maintain 2px stroke width for visual consistency
  • Match icon color to surrounding text or use brand colors
  • Ensure icons are appropriately sized for their context
  • Use icons to enhance, not replace, text when possible
  • Maintain consistent spacing around icons

‚úó Avoid

  • Changing stroke width from the standard 2px

  • Placing icons without adequate spacing

Imagery & Photography

Visual imagery plays a crucial role in conveying our brand message. Follow these guidelines for selecting and using images.

Image Style

  • Professional yet Approachable: Images should feel professional but not corporate or cold
  • Modern & Clean: Prefer clean, uncluttered compositions
  • Authentic: Use real, relatable imagery over overly staged photos
  • Bright & Optimistic: Choose images with good lighting and positive energy

Color Overlays

When using images with text overlays, apply subtle color overlays to ensure readability. Use the brand gradient as overlay or accent on images:

Brand Gradient Overlay

Dark Navy Overlay

Navy Blue Overlay

Image Composition

  • Use the rule of thirds for balanced compositions
  • Leave space for text overlays (especially in social media posts)
  • Avoid busy backgrounds that compete with text
  • Ensure faces and important elements are well-lit

Spacing & Layout Guidelines

Consistent spacing and layout create visual harmony and improve readability across all assets.

Spacing Scale

Use this 8px base unit system for consistent spacing:

8px - Tight spacing, icons
16px - Standard padding
24px - Comfortable spacing
32px - Section spacing
48px - Large gaps
64px - Major sections

Grid System

Use a 12-column grid system for layouts:

12-column grid with consistent gutters (16-24px)

Social Media Safe Zones

Keep important content within these safe zones to avoid platform UI elements:

Instagram Stories

  • Top 10%: Avoid (profile/close button)
  • Bottom 20%: Avoid (swipe up area)
  • Safe zone: Center 70% vertically

Facebook Posts

  • Top 15%: Avoid (profile/actions)
  • Bottom 10%: Avoid (engagement area)
  • Safe zone: Center 75%

Twitter/X Posts

  • Top 10%: Avoid (profile info)
  • Bottom 15%: Avoid (actions/engagement)
  • Safe zone: Center 75%

Text Alignment

Left Aligned

Use for most body text and content. This is the most natural reading flow and works well for longer content blocks.

Center Aligned

Use sparingly for headlines, quotes, or short statements. Avoid for long paragraphs.

Right Aligned

Use rarely, typically for specific design elements or when creating visual balance.

Visual Hierarchy Principles

  • Size: Larger elements draw attention first
  • Color: Use brand colors strategically to guide the eye
  • Contrast: High contrast creates emphasis
  • Whitespace: More space around elements increases importance
  • Position: Top-left and center are primary focal points
